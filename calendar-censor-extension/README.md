# Google Calendar Title Censor

Local Chrome extension that toggles solid black censor bars over visible Google Calendar event titles in the active tab.

## Load It

1. Open `chrome://extensions`.
2. Enable `Developer mode`.
3. Choose `Load unpacked`.
4. Select `/Users/linus/Documents/New project 2/calendar-censor-extension`.

## Use It

Open Google Calendar and click the extension icon to censor titles. Click the icon again to restore titles.

When censoring is active, the extension badge shows `ON`.

The extension only changes the page visually. Event titles remain in the page DOM, so this is intended for screenshots and screen sharing.
